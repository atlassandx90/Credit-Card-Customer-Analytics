# Credit Card Customer Analytics & Attrition Analysis

## Project Overview
This project analyzes customer behavior, attrition patterns, credit utilization, and customer segmentation using Power BI, SQL, and Python.

The objective of this project is to identify:
- Customer attrition trends
- Spending behavior
- Customer segmentation patterns
- Risk indicators related to credit card usage

The project combines:
- Interactive Power BI dashboards
- SQL-based data analysis
- Python exploratory data analysis (EDA)

# Key Business Insights

- Customers with lower income groups show higher attrition.
- Blue card holders dominate the customer portfolio.
- Higher credit utilization is associated with higher attrition risk.
- Middle-aged customers represent the largest customer segment.
- Married customers form a significant portion of the active customer base.
